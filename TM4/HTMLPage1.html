Class:- constructor type of constructor destructor ----------------------------------------------- Property:- A C# class variable also called property, to read and write value in property , we can use get and set keyword , called getter and setter. A class variable has by default no accessor. so if we want read only, we can use get keyword or read and write both we can use get/set both. ------------------------------------------------ Indexer:- A indexer is the virtual array, which allow to read and write class variables like array. A indexer permits get/set accessor. Note: a class can container more than one indexer , called index overloading. --------------------------------------
